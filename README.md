#  Web Projects Collection

###  Hotel Management System +  Student Admission Portal

This repository contains two interactive, front-end web applications built using **HTML, CSS, and JavaScript**. Both projects are designed to be dynamic (client-side) and visually engaging, with real-time updates and local storage support.

---

#  1. Hotel Management System

## Overview

The **Hotel Management System** is a dynamic web application that allows users to manage hotel room bookings, track availability, and calculate revenue in real time.

##  Features

*  Room availability tracking (Available / Booked)
*  Check-in & Check-out system
*  Automatic price calculation
*  Dashboard (Total rooms, booked, available, revenue)
*  Booking records table
*  Room checkout functionality
*  Modern UI with glowing effects

##  How It Works

* User enters guest details and booking dates
* System calculates total stay duration and cost
* Room status updates instantly
* Dashboard updates automatically

##  Technologies Used

* HTML5
* CSS3 (Flexbox, gradients, glassmorphism)
* JavaScript (DOM manipulation)

---

#  2. Student Admission Portal

##  Overview

The **Student Admission Portal** is a dynamic system that allows users to add, edit, delete, search, and filter student records with persistent storage.

##  Features

*  Add student details
*  Edit existing records
*  Delete students with confirmation
*  Search by name/email
*  Filter by course
*  Dark mode toggle
*  Live student counter
*  Data stored using LocalStorage
*  Responsive and modern UI

##  How It Works

* User fills the admission form
* Data is stored in browser (localStorage)
* Table updates dynamically
* Search and filters refine displayed data instantly

##  Technologies Used

* HTML
* CSS (Glass UI, transitions)
* JavaScript (LocalStorage, DOM manipulation)
